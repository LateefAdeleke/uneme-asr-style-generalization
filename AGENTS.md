Project rules:
- Read PROJECT_CONTEXT.md, README_EXPERIMENTS.md, and configs/experiment_registry.yaml before making changes.
- Do not modify dataset structure.
- Do not regenerate split files.
- Always use audio_path from metadata.
- Audio files are not included in this repository.
- Local audio roots are:
  - data/audio/
  - data/careful_audio/
- Scripts must validate audio file existence before training.
- Prefer small, modular changes over large refactors.
- Preserve experiment IDs exactly as defined in the registry.
- Main split is the primary benchmark.
- reverse_aux and mixed_to_constrained_aux are supplementary.
